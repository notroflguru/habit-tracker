# Habit Tracker

Консольное приложение для отслеживания привычек с хранением данных в PostgreSQL.

## О проекте
Это pet-проект, созданный для практики и освоения:
- Чистой архитектуры и разделения слоёв
- JDBC и SQL
- Работы с Docker и PostgreSQL

## Функционал
- Регистрация и авторизация пользователей
- Создание, просмотр и удаление привычек
- Привязка привычек к конкретному пользователю
- Валидация входных данных
- Хранение данных в PostgreSQL через JDBC

## Технологии
- **Java** 17
- **PostgreSQL**
- **JDBC**
- **Docker**
- **Maven**
